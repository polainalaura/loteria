Este proyecto consiste en el desarrollo de un programa en Java que simula el funcionamiento básico de una lotería. El programa genera automáticamente tres números premiados y permite al usuario comprobar si su décimo coincide con alguno de ellos.

El usuario puede introducir su número de décimo y el importe jugado para saber si ha ganado algún premio. Además, el programa permite consultar los números premiados generados en el sorteo.

El programa compara el número introducido con los números premiados generados aleatoriamente. Si coincide con alguno de ellos, se calcula el dinero ganado.

Si el número no coincide con ninguno de los premios, el programa muestra un mensaje indicando que no ha habido suerte.
